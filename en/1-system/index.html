设备按键键值（System KeyCode）

用户操作(Action) 简单功能描述(Result) 应用层键值（KeyCode） 注释（Notes）
Back-长
(Back long press)		 开启快捷菜单
(open shortcut)		 none 被系统占用
(System handled)
Back-单
(Back click)		 返回
(Back)		 KEYCODE_BACK = 4
TP-前滑
(swipe forward)		 应用可同时收到多个
KEYCODE_DPAD_RIGHT键值和单个
KEYCODE_DPAD_DOWN键值
(App can get severl KEYCODE_DPAD_RIGHT and one KEYCODE_DPAD_DOWN )		 KEYCODE_DPAD_DOWN = 20
KEYCODE_DPAD_RIGHT = 22		 希望焦点跳一个，使用DOWN；希望焦点跳多个，使用RIGHT，键值数与滑动距离正相关；
(Use DOWN for next focus; use RIGHT for severl focus switch, KEYCODE_DPAD_RIGHT number is releated to swipe length.)
TP-后滑
(swipe backward)		 应用可同时收到多个
KEYCODE_DPAD_LEFT键值和单个
KEYCODE_DPAD_UP键值
(App can get severl KEYCODE_DPAD_LEFT and one KEYCODE_DPAD_UP )		 KEYCODE_DPAD_UP = 19
KEYCODE_DPAD_LEFT = 21		 希望焦点跳一个，使用UP；希望焦点跳多个，使用LEFT，键值数与滑动距离正相关；
(Use UP for previous focus; use LEFT for severl focus switch; KEYCODE_DPAD_LEFT number is releated to swipe length.)
TP-单击
(TP tap)		 确认
(Confirm/Select)		 KEYCODE_DPAD_CENTER = 23
TP-长按
(TP-long press)		 KEYCODE_TV = 170 应用可自定义
(App handle)
Power 电源键
(turn on/off screen)		 KEYCODE_POWER = 26
