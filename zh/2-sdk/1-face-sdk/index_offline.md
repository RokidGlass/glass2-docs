# 人脸识别SDK


## 一. 人脸识别SDK介绍

### 1.1 概述
人脸识别SDK提供基础的人脸检测+人脸跟踪+人脸识别，能够高效进行多人识别。本SDK封装底层算法接口，提供：

1.图片人脸检测+图片人脸识别<br>
2.相机预览数据人脸检测，人脸跟踪，人脸识别。<br>
3.人脸数据库增删改查的接口<br>
4.能够获取人脸角度以及人脸质量等信息<br>
5.单帧图片人脸检测，支持bitmap、NV21格式数据人脸检测<br>
